Extraction project and data visualization coming from Tripadvisor

This website was created to display data collected by a scraping software (always created by me and you can consult it here) that was executed on board a Raspberry Pi 3B+ which collected data automatically for a period of one month from TripAdvisor. The website uses display technologies from data such as Highcharts and Tabulator and the data is directly loaded by a specific database. This entire project constitutes my thesis for the three-year degree in Computer Science and through this link you can consult the official documentation available only in Italian.
