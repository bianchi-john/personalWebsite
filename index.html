FAO-ASFA project
As part of the FAO-ASFA project, ISTI-CNR is engaged in the implementation of a text mining service for documents on fisheries, aquaculture and marine biodiversity. During the proposed internship, it was required to implement an algorithm for extracting categorized information from text, exploiting the Web service named NLPHub of ISTI-CNR. The algorithm was finally published in as-a-service mode on the CNR cloud computing platform named DataMiner, using the software import tools offered by the platform itself.

The work was carried out in collaboration with Dr. Gianpaolo Coro who led the project.

NLP with Disaster Tweets
Development of a machine learning model that predicts which Tweets are about real disasters and which are not.

Created for the "Text Analytics" exam, this project illustrates the realization of two different models both able to operate this classification task: the first one, based on a traditional approach, uses the technique of feature engineering while the second one uses the Bert language model through Simple Transformers library.

The work was carried out in collaboration with Dr. Andrea Esuli who has oversaw the project.

Virtual museum of music album covers

Finished in december 2020
Virtual museum of music album covers
This archive lets you explore the music albums through the colour of their covers. You can enjoy it by randomly exploring the visualizations or by looking at the specific categories. All the music data, including images, are provided by Spotify. The authors' contribution is relative to the chromatic analysis of the data and to their visual and logical organisation.

Website developed in collaboration with Marta Fioravanti
